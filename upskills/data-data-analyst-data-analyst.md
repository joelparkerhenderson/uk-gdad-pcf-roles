# Data analyst - upskill resources

This page focus on learning and growth resources for the work role of data analyst.

- The resources include links to online courses, blogs, research articles, videos, and books. 

- These help with job upskilling, work training, professional development, and career advancement,

- These are intended to be good starting points, not a comprehensive catalogue.

We welcome constructive feedback to improve this page and these lists.

- GitHub issues and/or pull requests are welcome.

- The resources are found via AI searches then go through human-in-the-loop proofing.

- For more contact Joel Henderson <joel@joelparkerhenderson.com>

- [IBM Data Analyst Professional Certificate](https://www.coursera.org/professional-certificates/ibm-data-analyst): This flexible online course prepares those without experience for entry-level positions through eight courses taking approximately 11 months to complete. - [Excel Skills for Data Analytics and Visualization Specialization](https://www.coursera.org/specializations/excel-data-analytics-visualization): Offered by Macquarie University, this specialization teaches how to bring data to life using advanced Excel functions, creative visualizations, and powerful automation features. ## Blog posts

- [Emerging Technologies and Applications in Data Analytics for 2025](https://www.ironhack.com/us/blog/data-analytics-in-2024-emerging-technologies-and-applications): This Ironhack blog post explores the latest trends in data analytics for 2025, including AI-powered automated cleaning tools like Trifacta and OpenRefine. The article covers natural language processing (NLP) and text mining for extracting insights from text data, using NLP tools like spaCy and NLTK for sentiment analysis and entity recognition. It discusses clustering techniques such as K-means, hierarchical clustering, and DBSCAN for market segmentation, fraud detection, and customer analytics.

- [Data Cleaning Best Practices: The Foundation for Reliable Reporting](https://databox.com/data-cleaning-best-practices): This Databox article discusses how data preparation goes beyond cleaning to include merging data from multiple sources, reshaping datasets, creating calculated fields, and filtering data to align with business needs. The blog emphasizes that over 80% of a data scientist's time is spent cleaning and organizing data, and that poor data quality costs organizations an average of $15 million annually.

- [5 Principles of Data Ethics for Business](https://online.hbs.edu/blog/post/data-ethics): Harvard Business School Online's blog provides career insights on data ethics, covering five guiding principles for business professionals who handle data. The article emphasizes that ideas around data ethics and best practices are constantly evolving, making it important for professionals to commit to continuous ethical education. Key principles include privacy protection, transparency, and consent.

- [Rebecca Barter's Blog - An Introduction to Python for R Users](https://rebeccabarter.com/blog/2023-09-11-from_r_to_python): This blog post introduces Python for data analysis from the perspective of an R (tidyverse) user. The post focuses on working with data in Python using pandas, helping R users transition to Python for data analysis tasks by drawing parallels between R and Python approaches.

- [SQL Queries for Data Analysis - LearnSQL.com](https://learnsql.com/blog/sql-queries-for-data-analysis/): This article demonstrates simple SQL queries that can be used for basic data analysis. LearnSQL also provides an advanced guide covering seven advanced SQL queries useful for data analysis. The blog covers essential SQL techniques that data analysts use daily for querying and manipulating data in relational databases.

- [GeeksforGeeks - SQL for Data Analysis](https://www.geeksforgeeks.org/sql/sql-data-analysis/): This comprehensive guide covers everything from basic SQL concepts to advanced techniques for data analysis. SQL (Structured Query Language) is described as a powerful tool for data analysis, allowing users to efficiently query and manipulate data stored in relational databases, whether working with sales, customer, or financial data.

- [Power BI Dashboard Tutorial - DataCamp](https://www.datacamp.com/tutorial/power-bi-dashboard-tutorial): DataCamp offers a step-by-step tutorial on creating Power BI dashboards, from loading datasets to sharing with teams. The tutorial guides readers through utilizing different design and format features available in Power BI to make dashboards stand out. It covers the three key components in creating a dashboard: Data (foundation), Design (intuitive and visually appealing layout), and Interactivity (allowing users to explore data using filters and slicers).

- [Tableau Dashboard Examples - DataCamp](https://www.datacamp.com/blog/tableau-dashboard-examples): DataCamp provides tutorials to build dynamic dashboards and create compelling stories in Tableau using real-world datasets. The blog includes essential principles of Tableau dashboard design for creating impactful visualizations. According to Andy Cotgreave, Technical Evangelist at Tableau, humans naturally look at things from left to right and top to bottom, which should inform dashboard design.

- [Data Storytelling - Sigma Computing](https://www.sigmacomputing.com/blog/stakeholder-data-communication): This blog focuses on making data insights clear and actionable for stakeholders through storytelling, visualizations, and effective communication strategies. The article emphasizes that different stakeholders need different levels of detail and process information in different ways, noting that what resonates with a data analyst won't necessarily work with a COO.

- [Introduction to Data Storytelling - Correlation One](https://www.correlation-one.com/blog/data-storytelling): This resource explains how data storytelling bridges the gap between technical and non-technical stakeholders, enabling informed decisions based on data-driven insights. It defines data storytelling as translating complex data analysis into compelling narratives that unlock insights, inform decision making, and drive action. Research shows that 71% of executives prioritize data storytelling skills for reporting to the C-suite, and data-driven stories can boost audience engagement by 300%.

## Research articles

- [Statistical Process Monitoring based on Functional Data Analysis (arXiv 2025)](https://arxiv.org/abs/2510.25742): This recent October 2025 paper by Fabio Centofanti focuses on statistical process monitoring using functional data analysis. The paper models profiles as functional data and applies FDA tools to address limitations of traditional monitoring techniques, offering modern approaches to quality control and process monitoring in data-intensive environments.

- [Doing Better Data Visualization (SAGE Journal 2021)](https://journals.sagepub.com/doi/full/10.1177/25152459211045334): Published by Hehman and Xie, this article emphasizes goal-centered graph design as a philosophy for making visualizations most effective. The research evaluates visualizations based on signal-to-noise ratio, where effective communication maximizes the signal-to-noise ratio by minimizing visual clutter. Data visualization methods have rapidly advanced over the past decade, with improvements in open-source software and computational power.

- [Making Data Visualization More Efficient and Effective: A Survey (VLDB Journal 2019)](https://link.springer.com/article/10.1007/s00778-019-00588-3): This survey article notes that data visualization is crucial in today's data-driven business world, with an emerging need for database experts to help with efficient and effective data visualization. The research provides comprehensive coverage of techniques and approaches for improving both the efficiency and effectiveness of data visualization systems.

- [Big Data Analytics using Machine Learning Techniques (IEEE)](https://ieeexplore.ieee.org/document/8776614/): This IEEE paper discusses challenges with using traditional machine learning techniques for big data analytics and proposes solutions including parallel processing, dimensionality reduction, GPUs, MapReduce, deep learning, and online learning. The paper addresses the intersection of machine learning and big data, providing practical approaches for analysts working with large-scale datasets.

- [Methodologies for Data Quality Assessment and Improvement (ACM 2009)](https://dl.acm.org/doi/10.1145/1541880.1541883): This foundational ACM Computing Surveys article provides a systematic and comparative description of data quality assessment methodologies. The paper compares methodologies along several dimensions, including methodological phases and steps, strategies and techniques, data quality dimensions, types of data, and types of information systems addressed by each methodology.

- [A Review of Data Quality Assessment Methods for Public Health Information Systems (PMC)](https://pmc.ncbi.nlm.nih.gov/articles/PMC4053886/): This review found that completeness, accuracy, and timeliness were the three most-used attributes among a total of 49 attributes of data quality. The major quantitative assessment methods were descriptive surveys and data audits, whereas the common qualitative assessment methods were interview and documentation review.

- [Organizational Business Intelligence and Decision Making Using Big Data Analytics](https://www.sciencedirect.com/science/article/abs/pii/S0306457321002090): This research addresses challenges including plan failure, lack of preparation, resource failure, and risk-taking capability, proposing frameworks to increase intelligent organizational effectiveness and decision-making analysis. The paper examines how organizations can leverage business intelligence and big data analytics to improve decision-making processes.

- [Predictive Big Data Analytics for Supply Chain Demand Forecasting (Journal of Big Data)](https://journalofbigdata.springeropen.com/articles/10.1186/s40537-020-00329-2): This survey investigates predictive big data analytics applications in supply chain demand forecasting, classifying algorithms into time-series forecasting, clustering, K-nearest-neighbors, neural networks, regression analysis, support vector machines, and support vector regression. The research provides practical guidance for data analysts working on forecasting problems.

- [From Data Mining to Knowledge Discovery in Databases (AI Magazine 1996)](https://ojs.aaai.org/aimagazine/index.php/aimagazine/article/view/1230): This foundational paper by Fayyad, Piatetsky-Shapiro, and Smyth provides an overview of the emerging field of data mining and knowledge discovery in databases, clarifying how these areas are related to each other and to related fields such as machine learning, statistics, and databases. This seminal work established many concepts still used in modern data analytics.

- [Exploratory Data Analysis Overview (Multiple Sources)](https://www.ncbi.nlm.nih.gov/books/NBK557570/): Exploratory data analysis (EDA) is an approach to analyzing datasets to summarize their main characteristics, often using statistical graphics and visualization methods. EDA is an essential step in any research analysis, with the primary aim being to examine the data for distribution, outliers, and anomalies. John W. Tukey wrote the foundational book "Exploratory Data Analysis" in 1977, establishing principles still used today.

## Books

- [Python for Data Analysis by Wes McKinney (3rd Edition, 2022)](https://www.amazon.com/Python-Data-Analysis-Wes-McKinney-ebook/dp/B0B9HY3WX7): Written by Wes McKinney, the creator of the Python pandas project, this book is ideal for analysts new to Python and for Python programmers new to data science and scientific computing. The book is concerned with the nuts and bolts of manipulating, processing, cleaning, and crunching data in Python, offering a comprehensive guide to becoming an effective data analyst. The latest edition was updated in 2022 with modern best practices.

- [Statistics for Data Science: A Beginner-Friendly Guide by Amrita Panjwani](https://www.amazon.com/Statistics-Data-Science-Beginner-Friendly-Concepts/dp/B0FK2MRVR8): A beginner-friendly guide to statistical thinking, practical concepts, and real-world code, written for professionals, students, and non-technical learners stepping into data-driven careers. The book includes Python code examples, distributions, probability, regression, and hypothesis testing. It's designed to make statistics accessible and practical for data science beginners.

- [Data Visualization with Tableau and Power BI by Thompson Carter (2024)](https://www.amazon.com/Data-Visualization-Tableau-Power-BI/dp/B0DHK9G9TS): This complete guide to becoming proficient in both Tableau and Power BI was published in September 2024. The book walks readers through each step, from setting up the software to building advanced dashboards and reports. It includes practical examples, real-world case studies, and tips from industry experts, covering everything from data storytelling fundamentals to advanced analytics.

- [Business Intelligence in Plain Language by Jeremy Kolb](https://www.amazon.com/Business-Intelligence-Plain-Language-practical-ebook/dp/B009K7INOY): This book provides a good high-level overview of business intelligence for non-technical readers and covers all major key terms and concepts needed to plan a BI solution. It serves as a practical guide to data mining and business analytics, making complex concepts accessible to business professionals.

- [Storytelling with Data by Cole Nussbaumer Knaflic](https://www.amazon.com/Storytelling-Data-Visualization-Business-Professionals-ebook/dp/B016DHQSM2): This bestselling book aims to give readers the skills and power to tell stories with their data. Cole Nussbaumer Knaflic, founder and CEO of Storytelling with Data, provides foundational lessons on effective data visualization and communication. The book has become widely used in business and academic settings for teaching data visualization principles.

- [Hands-On Data Analysis with Pandas by Stefanie Molin](https://www.amazon.com/Hands-Data-Analysis-Pandas-visualization/dp/1800563450): This book shows how to analyze data, get started with machine learning, and work effectively with Python libraries often used for data science, such as pandas, NumPy, matplotlib, seaborn, and scikit-learn. It uses real-world data with real-world issues and shows how to work with realistic datasets to master the use of pandas for data analysis.

- [Excel Data Analysis For Dummies by Paul McFedries](https://www.amazon.com/Excel-Data-Analysis-Dummies-McFedries-ebook/dp/B09Q879R62): This book covers how to use Excel's built-in data analysis features, create charts and visualizations, clean up large datasets, and perform financial calculations and database operations. The book teaches creating and visualizing data, deploying PivotTables, using statistical functions to predict future results, and more, making Excel accessible for data analysis tasks.

- [R for Data Science by Hadley Wickham, Mine Çetinkaya-Rundel, and Garrett Grolemund (2nd Edition)](https://www.amazon.com/R-Data-Science-Hadley-Wickham-ebook/dp/B0C7JKPC98): This book teaches aspiring data scientists how to do data science with R and RStudio, along with the tidyverse—a collection of R packages designed to work together. The second edition is a major reworking of the first edition, removing material no longer useful, adding new material, and updating text and code to reflect current best practices. The book is also available free online.

- [SQL for Data Analysis: Advanced Techniques by Cathy Tanimura](https://www.amazon.com/SQL-Data-Analysis-Techniques-Transforming/dp/1492088781): This practical book reveals ways to improve SQL skills and solve problems, teaching how to use SQL functions such as joins, window functions, subqueries, and regular expressions. The book provides advanced techniques for transforming data into insights, perfect for analysts looking to level up their SQL capabilities.

- [Data Mining: Concepts and Techniques by Jiawei Han, Micheline Kamber, and Jian Pei](https://www.amazon.com/Data-Mining-Concepts-Techniques-Management-ebook/dp/B0058NBJ2M): This comprehensive book provides concepts and techniques in processing gathered data and specifically explains data mining and the tools used in discovering knowledge from collected data. The book covers knowledge discovery from data (KDD) and is considered a foundational text in the field of data mining and analytics.

## Professional development tips

- Commit to continuous learning because tools and techniques evolve rapidly
- Build a portfolio of demonstration projects so you can share your skills
- Join relevant communities, attend events, and connect with colleagues
- Practice explaining concepts to audiences in person and also in writing
- Focus on developing both technical skills and business acumen
- Seek opportunities to work with diverse people and projects
